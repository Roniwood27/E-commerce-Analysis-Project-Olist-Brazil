# E-commerce-Analysis-Project-Olist-Brazil
In this project I cleaned the data, created data modeling, analyzed the data, and found some key insights in Excel.

E-commerce Analysis Project: Olist Brazil

Data Cleaning: Processed raw datasets by removing duplicates, handling missing values, and formatting dates to ensure accurate analysis.

Data Modeling: Connected different tables using a star schema in Power Pivot to make the data easy to filter and analyze.

Analysis & Key Insights

Sales Performance (2016-2018): Used pivot tables to determine that the business generated between $16M and $17M in total revenue.

Revenue by Top 10 Categories: Identified "Bed, Bath & Table" as the best-selling group among the top 10 categories, leading with $1.45M in sales.

Average Order Value: Calculated a typical spend of $154.10 per order, which helps define the platform's customer profile.

Customer Payment Method: Found that 74% of customers prefer to pay by credit card, highlighting the importance of financing options.

Visualization

Dashboard: Developed a simple dashboard to visualize sales trends, highlighting that November (Black Friday) reached the highest sales volume in the analyzed period.
